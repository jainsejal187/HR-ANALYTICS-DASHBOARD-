<h2 align : "center"><b>HR-ANALYTICS-DASHBOARD </b></h2>

<p align : "center">This project provides a comprehensive analysis of workforce data to uncover key insights on employee demographics, department performance, attrition patterns, job satisfaction, and skills distribution. Using descriptive statistics and data visualization, the analysis identifies trends such as mid-career workforce concentration, high attrition in R&D and Sales, and factors influencing employee satisfaction.</p>

<h3 align : "center"><b>Key Features:</b></h3>

<pre align : "center"> Workforce demographics and department-level insights 

Attrition analysis and job satisfaction trends

Identification of high-risk groups for retention

Actionable recommendations for employee retention, work-life balance, performance recognition, diversity, and skill development

The dashboard serves as a strategic tool for HR teams to improve employee engagement, retention, and organizational performance.</pre>
